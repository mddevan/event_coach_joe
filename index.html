Potomac Lanes Presents COACH JOE SLOWINSKI.

By joining this event you will learn, Where to play when you don’t know the pattern and Where to play when you have the pattern. You will also be taught How lane surfaces affect, Pattern Manipulation and How to Interpret a Pattern Graph.

You will gain knowledge about the complex variables that create a world-class release, Drills to improve the game and how to warm up for bowling and why. There will be also a module that teaches a spare shooting system that is applied to any pattern.

JOIN US FOR A DAY OF TRAINING WITH USBC GOLD COACH JOE SLOWINSKI!

Module Details

Here you will find more details about every module.

Module 1 – Lane Play Strategies

  • Where to play when you don’t know the pattern details
  • Where to play when you have the pattern details
  • How lane surfaces affect where I play (higher friction environments i.e. HPL, wood)
  • Pattern Manipulation
  • Interpret a Kegel & Brunswick Pattern Graph
  • Practice interpreting many pattern graphs.
  • Make lane play plans while bowling on multiple patterns

Module 2 - Release Development & Practice Drills

  • Learn the complex variables that create a world-class release
  • Training Drills to improve your game
  • Warming up for bowling
  • Learn a dynamic warm-up for bowling
  • Learn physical game drills to improve your physical game
  • Develop release variation while doing physical game drills
  • Implement the physical game drill process

Module 3 – Improve Your Spare Shooting

  • Learn how to become an advanced spare shooter
  • Implement the spare shooting system on multiple patterns
